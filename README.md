# jpa-hibernate-data-access

Projeto demonstrando **acesso a dados em Java utilizando JPA (Java Persistence API) com Hibernate
---

## Descrição

Este projeto tem como objetivo demonstrar:

- Configuração de JPA com Hibernate
- Mapeamento de entidades
- Operações básicas de CRUD (Create, Read, Update, Delete)
- Gerenciamento de transações
- Configuração de persistência via `persistence.xml`

---

## Tecnologias Utilizadas

- Java
- Maven
- JPA (Java Persistence API)
- Hibernate ORM
- Banco de dados relacional (mySql)
